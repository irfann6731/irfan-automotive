# Irfan Automotive

Irfan Automotive merupakan platform web layanan bengkel sepeda motor yang berfokus pada kemudahan akses informasi servis dan transparansi harga bagi pengguna di wilayah Yogyakarta. Website ini dirancang untuk memberikan pengalaman pengguna yang profesional dan efisien dalam melakukan pemesanan layanan otomotif.

## Deskripsi Proyek

Proyek ini bertujuan untuk menyediakan media informasi digital bagi pemilik kendaraan roda dua yang membutuhkan layanan perawatan rutin maupun perbaikan komponen. Dengan antarmuka yang modern dan responsif, pengguna dapat melihat berbagai kategori layanan, estimasi biaya, serta testimoni dari pelanggan lainnya secara langsung melalui perangkat komputer maupun seluler.

## Fitur Utama

1. Katalog Layanan: Informasi mengenai layanan servis seperti penggantian oli, penggantian ban, dan servis rutin berkala.
2. Daftar Harga Transparan: Rincian biaya untuk setiap paket layanan guna memberikan kepastian harga bagi calon pelanggan.
3. Sistem Konfirmasi Pesanan: Integrasi pembayaran menggunakan sistem QRIS dan konfirmasi langsung melalui aplikasi WhatsApp untuk mempermudah transaksi.
4. Desain Responsif: Pengoptimalan tampilan menggunakan kerangka kerja Bootstrap sehingga website tetap fungsional di berbagai ukuran layar.

## Layanan Unggulan

Berikut adalah beberapa layanan utama yang tersedia pada platform ini:

1. Ganti Oli: Layanan penggantian pelumas mesin menggunakan produk berkualitas premium.
2. Ganti Ban: Tersedia berbagai pilihan ban dengan jaminan kualitas dan ketahanan tinggi.
3. Service Rutin: Pemeriksaan menyeluruh terhadap seluruh komponen kendaraan oleh tenaga mekanik profesional.

## Teknologi yang Digunakan

Pengembangan website ini memanfaatkan teknologi standar industri web sebagai berikut:

1. HTML5: Digunakan untuk membangun struktur dasar halaman web.
2. CSS3: Digunakan untuk penataan gaya visual, termasuk implementasi kustomisasi warna dan tipografi.
3. Bootstrap 5: Digunakan sebagai kerangka kerja CSS utama untuk mempercepat pengembangan tata letak yang responsif.
4. Bootstrap Icons: Digunakan untuk penyediaan aset ikonografi pada antarmuka.
5. JavaScript: Digunakan untuk logika interaksi pada elemen modal dan fungsionalitas pendukung lainnya.

## Instruksi Penggunaan

Untuk menjalankan proyek ini di lingkungan lokal, Anda dapat mengikuti langkah-langkah di bawah ini:

1. Unduh atau klon repositori ini ke penyimpanan lokal Anda.
2. Pastikan seluruh aset pada direktori assets tersedia dalam struktur folder yang benar.
3. Buka berkas index.html menggunakan peramban web pilihan Anda.
4. Pastikan koneksi internet tersedia untuk memuat dependensi eksternal seperti Bootstrap Icons dari CDN.
